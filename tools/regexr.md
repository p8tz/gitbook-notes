## 正则

### 元字符

```bash
^	# 从文本首部开始匹配	使用m开启按行首匹配
$	# 从文本尾部开始匹配	使用m开启按行尾匹配
.	# 匹配任意一个字符
()	# 标记为一个表达式
*	# 匹配前一个表达式0到多次
+	# 匹配前一个表达式1到多次
?	# 匹配前一个表达式0到1次
{m}	# 匹配前一个表达式m次
{m,}  # 匹配前一个表达式至少m次
{m,n} # 匹配前一个表达式m到n次
.*	# 匹配任意
[]	# 匹配其中一个
[^]	# 不匹配其中任意一个
\b	# 匹配单词边界	 比如\babc不能匹配   xabc
\B	# 匹配单词非边界	 比如\Babc不能匹配	abc
abc|def	# 匹配abc或def
```

```bash
\w	# 匹配数字、字符、下划线, 相当于[a-zA-Z0-9_]
\W	# 匹配非数字、字符、下划线, 相当于[^a-zA-Z0-9_]
\d	# 匹配数字, 相当于[0-9]
\D	# 匹配非数字, 相当于[^0-9]
```

### 修饰符

```bash
/^Chapter/g	# 后面的g就是修饰符
g	# 全局匹配
i	# 不区分大小写
m	# 多行匹配, 使^和$匹配每一行的开头和结尾
s	# 默认.不会匹配换行符, 加上.后会匹配换行符
```
